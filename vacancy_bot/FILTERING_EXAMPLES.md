# 📋 Фильтрация низкокачественных вакансий

## 🎯 Обзор

Реализована строгая фильтрация сомнительных и низкокачественных вакансий с использованием **scoring-системы**.

## 🔍 Логика фильтрации

### Шаг 1: Жесткая блокировка (стоп-фразы)
Если найдена хотя бы одна стоп-фраза → вакансия **отклоняется немедленно**.

### Шаг 2: Scoring-система
Подсчитываются баллы за наличие важных элементов:
- **+1 балл** за название должности
- **+1 балл** за стек/навыки
- **+1 балл** за формат работы (remote/full-time)
- **+1 балл** за внятное описание задач
- **+1 балл** за профессиональные задачи

### Шаг 3: Вычитание баллов
Вычитаются баллы за подозрительные элементы:
- **-2 балла** за "без опыта"
- **-2 балла** за заработок "от N в день"
- **-2 балла** за опросы/отзывы/установки
- **-2 балла** за отсутствие работодателя
- **-2 балла** за контакт через бота

### Шаг 4: Финальная оценка
Если **итоговый score < 0** → вакансия **отклоняется**.

## ❌ Примеры отклоняемых вакансий

### Пример 1: Стоп-фраза "без опыта"
```
Ищем сотрудника без опыта работы. 
Заработок от 500₽ в день. 
Пишите в ЛС.
```
**Причина отклонения:** Стоп-фраза "без опыта"

---

### Пример 2: Стоп-фраза "заработок от N в день"
```
Работа с телефона. 
Заработок от 1000₽ в день. 
Свободный график.
```
**Причина отклонения:** Стоп-фраза "заработок от 1000₽ в день"

---

### Пример 3: Стоп-фраза "опрос"
```
Нужны люди для прохождения опросов. 
Оплата за каждый опрос. 
Пишите в ЛС.
```
**Причина отклонения:** Стоп-фраза "пройти опрос"

---

### Пример 4: Стоп-фраза "установить приложение"
```
Установите приложение и получите деньги. 
За каждую установку - оплата. 
Детали в ЛС.
```
**Причина отклонения:** Стоп-фраза "установить приложение"

---

### Пример 5: Низкий score (нет важных элементов)
```
Нужны люди. 
Работа удаленно. 
Пишите в ЛС.
```
**Причина отклонения:** 
- Score: -1 (нет должности: -2, нет стека: -2, нет задач: -2, есть формат работы: +1)
- Причины: нет названия компании, нет названия должности

---

### Пример 6: Контакт через бота
```
Ищем дизайнера. 
Напишите боту @somebot для деталей.
```
**Причина отклонения:** Стоп-фраза "бот для заработка" или контакт через бота

---

### Пример 7: Массовый набор
```
Массовый набор в команду. 
Нужны все желающие. 
Без ограничений.
```
**Причина отклонения:** Массовый набор (шаблон)

## ✅ Примеры проходящих вакансий

### Пример 1: Качественная вакансия
```
Ищем Python-разработчика в команду.

Требования:
- Опыт работы с Python 3+ (от 2 лет)
- Знание Django/FastAPI
- Опыт работы с PostgreSQL
- Git, Docker

Задачи:
- Разработка backend API
- Оптимизация производительности
- Code review

Условия:
- Удаленно (full-time)
- Зарплата: от 150000₽
- Компания: TechStart

Откликнись с портфолио!
```
**Score:** +5 (должность: +1, стек: +1, формат: +1, задачи: +1, описание: +1)
**Результат:** ✅ Проходит фильтр

---

### Пример 2: Вакансия с минимальными требованиями
```
Нужен дизайнер для создания логотипа.

Требования:
- Опыт работы с Figma
- Портфолио

Задача: Разработать логотип для стартапа.

Формат: Удаленно, проект.

Бюджет: 30000₽

Напишите в комментариях.
```
**Score:** +3 (должность: +1, стек: +1, задачи: +1)
**Результат:** ✅ Проходит фильтр (score >= 0)

## 📊 Структура данных

После фильтрации вакансия содержит флаг:
```python
vacancy = {
    "text": "...",
    "category": "it",
    "is_vacancy": True,      # Всегда True после проверки _is_vacancy()
    "is_suspicious": False   # Всегда False после проверки _is_suspicious_vacancy()
}
```

---

## 🔧 Конфигурация

Все стоп-фразы и паттерны находятся в `config.py`:
- `SUSPICIOUS_VACANCY_PHRASES` - жесткие стоп-фразы
- `MISSING_COMPANY_PATTERNS` - паттерны для определения компании
- `MISSING_POSITION_PATTERNS` - паттерны для определения должности
- `PROFESSIONAL_TASK_PATTERNS` - паттерны профессиональных задач
- `SKILLS_PATTERNS` - паттерны стека/навыков
- `WORK_FORMAT_PATTERNS` - паттерны формата работы
- `TASK_DESCRIPTION_PATTERNS` - паттерны описания задач
- `BOT_CONTACT_PATTERNS` - паттерны контакта через бота
- `MASS_RECRUITMENT_PATTERNS` - паттерны массового набора

---

## 📝 Логирование

Все отклоненные вакансии логируются с указанием причины:
```
🚫 Отклонена низкокачественная вакансия: Стоп-фраза: без опыта
Текст: Ищем сотрудника без опыта работы...
```

Или для scoring:
```
🚫 Низкокачественная вакансия: score=-1, причины: нет названия компании, нет названия должности
```
