# Feedback Bot - Бот для сбора отзывов

Бот для сбора отзывов о FreelanceHub. Пользователи оставляют отзывы, которые проходят модерацию. За одобренный отзыв начисляется 3 дня x2 статуса.

## Установка

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

2. Скопируйте `.env.example` в `.env` и заполните настройки:
```bash
cp .env.example .env
```

3. Выполните миграцию БД:
```sql
-- Выполните файл vacancy_bot/database/migration_add_reviews.sql в Supabase SQL Editor
```

4. Запустите бота:
```bash
python main.py
```

## Функционал

- Пользователи могут оставлять отзывы через команду `/review`
- Отзывы проходят модерацию в админ-панели основного бота
- За одобренный отзыв начисляется 3 дня x2 статуса
- Пользователи получают уведомления о статусе модерации

## Команды

- `/start` — Главное меню
- `/review` — Оставить отзыв
- `/help` — Справка
